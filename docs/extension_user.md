# GET

## user ()

Gets all users.

Returns a list of users, or an error response.

## user/{id} (id)

Get a user by id.

```
optional String id:     The user's id.
```

Returns the user, or an error response.

## user/{name} (name)

Get a user by name.

```
optional String name:     The user's name.
```

Returns the user, or an error response.
